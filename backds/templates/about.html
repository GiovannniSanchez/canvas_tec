Descripción de la innovación

La novedad de este proyecto es el diseño de un software con el uso de inteligencia artificial para simplificar el proceso de creación de modelos de negocio, lo que reduce costos y tiempo. Esta herramienta es accesible para personas sin conocimientos especializados, lo que fomenta el emprendimiento y la creación de empleos. Sabemos que generar un modelo de negocios puede ser muy costoso y complejo tanto para las empresas como para los estudiantes, sin embargo, para una empresa sino lo realiza de manera correcta es posible que la idea del negocio fracase. Al reducir costos y que sea de una manera mas sencilla podemos lograr una buena planeación de nuestra idea lo que nos lleve al éxito de esta y tener mayores oportunidades dentro del sector laboral. La aplicación de esta propuesta se contextualiza en el desarrollo tecnológico de una innovación tecnológica radical. El uso de inteligencia artificial para la creación de modelos de negocio es un cambio significativo en la forma en que se han creado históricamente estos modelos y al utilizar tecnología de vanguardia para abordar un problema empresarial común, se puede lograr una ventaja competitiva significativa para las empresas y proyectos estudiantiles que utilicen esta herramienta.
